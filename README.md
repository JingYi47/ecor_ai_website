# Ecor AI Website – Backend

## 1. Giới thiệu

Backend của hệ thống **Ecor AI Website** được xây dựng bằng **Node.js + Express + MongoDB**, cung cấp các RESTful API cho các chức năng:

* Quản lý người dùng (User)
* Sản phẩm (Product)
* Giỏ hàng (Cart)
* Checkout / Order
* Thanh toán (VNPay, MoMo)
* Review sản phẩm

---

## 2. Công nghệ sử dụng

* Node.js
* Express.js
* MongoDB + Mongoose
* JWT Authentication
* VNPay Payment Gateway
* MoMo Payment Gateway

## 4. Cài đặt project

### Clone project

```bash
git clone https://github.com/JingYi47/ecor_ai_website.git
cd backend
```

### Cài dependencies

```bash
npm install
```

### Tạo file `.env`

```env: liên hệ 
```

---

## 5. Chạy server

```bash
npm start/ npm run dev
```

Server chạy tại:

```
http://localhost:8000
```
